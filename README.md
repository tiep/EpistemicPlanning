# EpistemicPlanning
This README file is under constructing.

Solver for Epistemic Planning (tested on iOS 10.11.x and 10.12.x)

1. Compilation

        make

    + Requirements:

        flex, bison and a C++ compiler such as g++.

2. To run the solver:
  make
  
  ./cpa+ inputFileName
  
  E.g., ./cpa+ Experiment/Assemble_C.txt
